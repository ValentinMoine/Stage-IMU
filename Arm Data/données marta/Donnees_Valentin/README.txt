Les donnés que tu as 
Pour le Xsens tu as les classeur. Le nom indique:
la date, l'heure, le temps du début d'enregistrement et le type de mouvement. 
Pour le Vicon, ce sont des fichiers Excel qui s'appelent Marta_trial, puis le type de mouvement et l'ordre.



Comment les marquers sont mises
Les marquers Xsens étaient placés de la manière suivante:
marquers #1 - Xsens DOT 1 - épaule
marquers #2 - Xsens DOT 2 - coude
marquers #3 - Xsens DOT 3 - poignet
///
!!! FAIS ATTENTION COMMENT les marquers étaient placés sur le bras. La partie où je mets le chargeur regardait en haut. 
J'ai envoyé les photos avec l'email. 
///
Le marquers Vicon étaient placés SUR le marquer Xsens (voir les photos). Le nom de marquer est indiqué dans le nom de la colonne.



Le décalage du temps entre les donnés Vicon et Xsens.
Pour chaque essaie au début je lançais le Xsens, j'attendais que les marquers soit synchronisés avec le portable
et enfin je lançais l'enregistrement de Vicon. Une fois que 60 secondes d'enregistrement de Vicon sont términés (automatique), j'arrêtais
le Xsens.
Qu'est que cela signifie? Ca veut dire que le fichier Vicon ne dure que 60 secondes, cependant le Xsens commence une petite 
seconde AVANT le Vicon et prend sa fin une petite seconde APRES le Vicon.
